# IPコネクティビティ

## 概要
IPコネクティビティは、ネットワーク層（Layer 3）での接続性に焦点を当てています。このセクションでは、ルーティングの基本概念、IPアドレッシング、サブネット化、ルーティングプロトコルについて学びます。
